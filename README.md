# AI Sales Bot (Telegram + OpenAI)

Умный бот-продавец с веб-админкой для управления базой знаний.

## 🚀 Возможности
- **Умные ответы**: Использует OpenAI (GPT-4o/GPT-3.5) для общения.
- **База знаний**: Легко редактируется через веб-интерфейс.
- **Сбор контактов**: Автоматически сохраняет имена и телефоны клиентов.
- **Админ-панель**: Просмотр лидов и настройка промптов.

## 🛠 Установка и запуск

### 1. Настройка
В файле `.env` укажите ваши ключи:
```ini
TELEGRAM_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_ключ_openai
```

### 2. Запуск (Docker)
Самый простой способ — использовать Docker Compose.
```bash
docker-compose up -d --build
```
Бот запустится в фоне.

### 3. Доступ
- **Админ-панель**: `http://ваш-ip:8501`
- **Бот**: В Telegram

## 📂 Структура
- `bot.py` — Основной файл бота.
- `brain.py` — Логика ИИ.
- `admin.py` — Веб-интерфейс (Streamlit).
- `knowledge.json` — База знаний (тексты, цены).
- `users.json` — База данных пользователей (создается автоматически).

## 🆘 Полезные команды
- Перезагрузить бота: `docker-compose restart`
- Смотреть логи: `docker-compose logs -f`
- Остановить: `docker-compose down`
